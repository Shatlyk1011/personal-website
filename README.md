# 🌐 Personal Website

A modern personal portfolio website featuring custom animations powered by GSAP and Motion.
Built to showcase projects, ideas, and experiments with a strong focus on smooth user experience.

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](./LICENSE.md)

---

## ✨ Features

- Custom animations using GSAP and Motion
- Smooth scroll powered by Lenis
- Video showcase via Mux Player
- Scheduling integration with Cal.com
- Contact form with Zod validation
- Clean and minimal design
- Fully responsive layout
- Fast performance with Next.js Turbopack

---

## 🛠 Tech Stack

| Category    | Technology                              |
| ----------- | --------------------------------------- |
| Framework   | Next.js 15 (App Router)                 |
| Language    | TypeScript                              |
| Styling     | Tailwind CSS v4                         |
| Animation   | GSAP, Motion                            |
| Scroll      | Lenis                                   |
| Video       | Mux Player                              |
| Scheduling  | Cal.com Embed                           |
| Validation  | Zod                                     |
| HTTP Client | Axios                                   |
| Linting     | ESLint                                  |
| Formatting  | Prettier                                |

---

## 🚀 Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/Shatlyk1011/personal-website.git
cd personal-website
pnpm install
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
