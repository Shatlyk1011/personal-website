# 🌐 Personal Website

A modern personal website featuring custom animations powered by GSAP and Motion.  
Built to showcase projects, ideas, and experiments with a strong focus on smooth user experience.

---

## ✨ Features

- Custom animations using GSAP and Motion
- Clean and minimal design
- Fully responsive layout
- Fast performance
- Easy content updates

---

## 🛠 Tech Stack

- Animation: GSAP, Motion
- Framework: Next.js
- Styling: Tailwind CSS

---

## 🚀 Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/Shatlyk1011/personal-website.git
cd personal-website
pnpm install
pnpm run dev
